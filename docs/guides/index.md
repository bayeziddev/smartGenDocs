# Guides Overview

Welcome to the **Guides** section of SmartGen Docs. This section provides practical, step-by-step instructions and in-depth explanations to help you effectively use and customize SmartGen Docs for your documentation needs. Whether you're setting up your first project, configuring advanced features, or extending its capabilities, these guides are designed to walk you through the process.

## In This Section

This section covers the following essential guides:

*   **[Configuration Guide](configuration.md)**: Learn how to set up and customize your `smartgen.yml` file to define site metadata, navigation, and theme settings.
*   **[Theming Guide](theming.md)**: Understand how to customize the visual appearance of your documentation site using Jinja2 templates and CSS.
*   **[CLI Reference](cli.md)**: A comprehensive reference for all Command-Line Interface (CLI) commands, their options, and usage examples.
*   **[Autodoc Guide](autodoc.md)**: Detailed instructions on how to use the `autodoc` feature to automatically generate API reference documentation from your Python code.
*   **[Deployment Guide](deployment.md)**: Step-by-step instructions for deploying your generated static documentation site to various hosting platforms.

## Getting Started with Guides

If you're new to SmartGen Docs, we recommend starting with the [Configuration Guide](configuration.md) to understand how to set up your project's basic structure and settings. From there, you can explore other guides based on your specific requirements.

## Contributing to Guides

We welcome contributions to improve and expand our guides. If you have a tip, a common use case, or a detailed explanation that would benefit other users, please consider contributing to the project. Refer to the [Contributing Guide](../contributing.md) for more information.
