# Welcome to SmartGen Docs

Welcome to the official documentation for **SmartGen Docs**. This platform is designed to help you build, manage, and deploy professional-grade documentation with ease.

---

## What is SmartGen Docs?

SmartGen Docs is a lightweight, open-source static site generator. It transforms your Markdown files into a premium, responsive documentation website, allowing you to focus on content while we handle the presentation.

## Key Features

*   **Premium UI/UX**: Includes dark/light mode, responsive layouts, and modern aesthetics.
*   **Markdown-Centric**: Write in simple, clean Markdown.
*   **Automated Deployment**: Seamless CI/CD integration with GitHub Pages.
*   **API Auto-Gen**: Generate API references directly from your Python docstrings.
*   **Upload Manager**: A built-in web interface for non-technical content management.

## Get Started in Seconds

If you are new to the platform, check out our **[Getting Started](https://github.com/bayeziddev/smartGenDocs/blob/main/docs/getting-started.md)** guide to initialize your first project.
